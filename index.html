Sistema de Relatórios — Guarda Nacional

Para ligar o sistema ao Google Drive, necessita de um Google Client ID. Siga os passos abaixo (feito uma única vez pelo administrador):

  1. Aceda a console.cloud.google.com e crie um novo projeto (ex: "FACV Relatorios")

  2. Em APIs & Serviços → Biblioteca, ative a Google Drive API

  3. Em APIs & Serviços → Credenciais, clique em Criar Credenciais → ID de cliente OAuth 2.0

  4. Tipo: Aplicação da Web. Em Origens JavaScript autorizadas, adicione o URL do seu site GitHub Pages (ex: https://usuario.github.io)

  5. Copie o ID do cliente e cole abaixo
